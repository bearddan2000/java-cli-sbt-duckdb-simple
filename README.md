# java-cli-sbt-duckdb-simple

## Description
Reads from csv and json giving field names.

## Tech stack
- java
- sbt
  - duckdb

## Docker stack
- hseeberger/scala-sbt:11.0.2-oraclelinux7_1.3.5_2.12.10

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
